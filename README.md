# 🏡 Rentify – Airbnb Clone

Rentify is a full-stack rental web application inspired by Airbnb. It allows users to explore property listings, create accounts, add properties, leave reviews, and view locations on an interactive map.

🔗 Live Demo: https://wanderlust-rental-web-application.onrender.com/listings

---

## 🚀 Features

- 🔐 User Authentication (Signup/Login/Logout)
- 🏠 Create, Edit & Delete Listings
- ⭐ Review & Rating System
- 🗺️ Interactive Map Integration (Leaflet + OpenCage API)
- ☁️ Image Upload & Storage using Cloudinary
- 🔎 Search Listings by Title & Location
- 🛡️ Role-Based Authorization
- 📱 Fully Responsive Design

## ⚙️ Installation & Setup

1. Clone the repository

git clone https://github.com/Shubhamparbhane18/Rentify-Airbnb.git

2. Navigate to project folder

cd Rentify-Airbnb

3. Install dependencies

npm install

4. Create a `.env` file and add:

CLOUDINARY_CLOUD_NAME=your_cloud_name  
CLOUDINARY_KEY=your_key  
CLOUDINARY_SECRET=your_secret  
MAP_API_KEY=your_opencage_key  
DB_URL=your_mongodb_connection_string  
SESSION_SECRET=your_secret_key  

5. Start the server

npm start
